# Star Wars Holocron

A simple React application to explore the Star Wars universe using SWAPI (The Star Wars API).

## Features
- View all 6 original Star Wars films
- Animated opening crawl for each film
- Search and list Characters, Planets, Starships, Vehicles, and Species
- Fully responsive design using Tailwind CSS
- Data caching with TanStack Query

## Developing
This project uses Vite. To run locally:

```bash
npm install
npm run dev
```
